# PatientDedupe

Live status page for PatientDedupe, a fast and explainable patient record-matching
service with a C++ matching core, SQL blocking, a FHIR Patient/$match API, a human
review console, and Hadoop and Hive analytics.

This Space is synced automatically from the GitHub repository on every push to main.
Source: https://github.com/royalgillz/PatientDedupe
