# PatientDedupe - Matching playground

Compare two patient records and watch the match score update live. The scoring is the
project's C++ matching engine compiled to WebAssembly, running entirely in your
browser, so nothing you type leaves the page.

This Space is built from the Vite + React frontend and deployed automatically from
the GitHub repository on every push to main.
Source: https://github.com/royalgillz/PatientDedupe
